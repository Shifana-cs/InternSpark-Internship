# Fake News Detection using Deep Learning (LSTM)

## Objective

Build a Deep Learning model to classify news articles as Fake or Real.

## Dataset

Fake and Real News Dataset

## Technologies Used

* Python
* TensorFlow
* Keras
* Pandas
* NumPy
* Matplotlib

## Model

* Tokenization
* Text Preprocessing
* LSTM Neural Network

## Results

* Training Accuracy: 99.2%
* Validation Accuracy: 98.4%
* Test Accuracy: 98.15%

## Deliverables

* Jupyter Notebook
* Saved Model (.h5)
* Screenshots
* Report
